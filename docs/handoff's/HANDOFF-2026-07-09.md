  commit above plus the VAD batch (25b0d5a) may already feel fine.
- **"Hear other apps while on call": RESOLVED on this machine.** Root
  cause was Windows communications ducking (default: reduce other sounds
  80% when the call's getUserMedia stream opens). Set
  `HKCU\Software\Microsoft\Multimedia\Audio\UserDuckingPreference = 3`
  ("Do nothing"; revert via Sound Control Panel → Communications tab).
  Follow-up candidate: a Voice setting that writes this value for other
  installs.
